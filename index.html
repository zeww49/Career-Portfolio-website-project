1) CSS Box Model

This card shows margin (space outside), border (outline), and padding (space inside).

Box Model Card

Margin creates space outside this card. The border outlines the card. Padding creates space between the border and this text.

3) Advanced CSS Image Example

This image uses clip-path (advanced CSS) to crop the image into a custom shape.

Why this is “advanced”

clip-path lets you mask/crop an image into shapes that are not rectangles (like polygons or circles). This is considered an advanced CSS image technique.
